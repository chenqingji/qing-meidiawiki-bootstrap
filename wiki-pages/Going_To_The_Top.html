To the Top

A very basic style that allows the visitor to scroll to the top of the page with a single click of the mouse, which can be useful for large articles. To the Top is not available in 1.0.43, but can be added using the steps below.

Install

Add the following to the bottom of bootstrap/bootstrap/bootstrap-blue.css (or your chosen color). 

.going-top:hover {
    text-decoration: none;
    background: none repeat scroll 0% 0% #555;
    border-bottom: 2px solid #333;
    color: #EEE;
    transition: all 0.3s ease-in-out 0s;
}
.going-top {
    padding: 3px 3px 3px 10px;
    background: none repeat scroll 0% 0% #EEE;
    border-bottom: 2px solid #CCC;
    color: #555;
    font-size: 1em;
    border-radius: 4px;
    margin: 1em auto 0px;
    float: right;
    transition: all 0.3s ease-in-out 0s;
}

Next create an id for the location where you want To the Top to scroll to when clicked. In our example we use the h3 panel class, e.g. below id="top, but you can use any phrase so long as it matches the link id. Where you scroll to is your choice and you can use any element to add an id i.e. <p id="top">

To the Top

And finally add the following when ever you want to scroll to the top.

[[#top|TOP]]
