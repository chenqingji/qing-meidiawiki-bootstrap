Font Awesome

Font Awesome extension for Vector.

Use any the font awesome icons shown [http://mediawikibootstrapskin.9ong.com/index.php?title=Icons_Sets here].

Installation

Download and unpack into your extensions folder and add the following to the bottom of LocalSettings.php

require_once "$IP/extensions/BootStrapSkinAwesome/awesome.php";

Vector Use

This extension was created for Vector - Font Awesome is included with every version of Bootstrapskin.
