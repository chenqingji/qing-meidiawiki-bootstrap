General Questions

Why does Bootstrapskin seem slow to load?
×

Bootstrapskin Speed!

Bootstapskin isn't slow, though this website is!

Our website is slow because we demonstrate the entire library of Bootstrapskin products, which amounts to a whole heap of data for a browser to load. To get a fair comparision visit a website that uses Bootstrapskin but only uses a moderate quantity of extensions.

Websites that only use a fair amount of extensions (perhaps a dozen or so) run upto 10 times faster visit [http://radxa.com/Home radxa.com] to see the difference, or [http://crpscommunity.co.uk/ crpscommunity.co.uk] which only uses three extensions.

Bootstrapskin Style

Bootstrapskin is packed with as many features and styling variations as we can get away with. Our goal has always been to ensure that websites that use our software do not look the same and to achieve this means including plenty of variation, hence the overly large bootstrap.css.

The main Bootstrapskin style is 770 kb, if you want to reduce this visit a [http://csscompressor.com/ css minifying service], this will reduce the style by 25%, resulting in a further increase in page load - with a potential 1 second page load.

Why was Bootstrapskin Created?
×

Why we created Bootstrapskin

The goal of Bootstrapskin is to make Mediawiki more accessible to new developers who want to learn about Mediawiki, how to create new skins and Mediawiki extensions.

There are several tutorials to get you started, and include how to modify Bootstrapskin and create a custom skin of your own, how to create Mediawiki Special pages and how to create a basic extension.

See the Bootstrapskin/Tutorial menu item for details.

What does Open Source mean?
×

Free to use sofware

In short it means you can download the software and use it exactly as you wish!

Though we always appreciate it when users keep the footer icon, you are not under any obligation to do so. We've even seen sites using Bootstrapskin, passing the work off as their own, perhaps not ethical but that's ok too

The only legal restriction is that you keep the licence intact (found in the root folder) which should always remain within the source files, even if you sell the software to someone else.

Frequent Problem Questions

Bootstrapskin doesn't work!
×

Fact Finding

It is very rare that Bootstrapskin doesn't work but the following may help to resolve your difficulties.

  • Revisit the installation guide and check that you have installed Bootstrapskin correctly, even the smallest deviation will cause problems. Errors are usually in LocalSettings.php so start there.
  • Bootstrapskin does not work with versions <1.22.5
  • If you're using any 3rd party extensions switch them off one at a time.
  • In the case of error messages, change the default template to Vector. Does this still result in an error message?
Getting Support

If after reviewing the installation, you still think there is an error, contact us at the support forums, making sure that you include the following in your message body.

  • Mediawiki version
  • Exact PHP version i.e. 5.?.?
  • List of third party extensions
