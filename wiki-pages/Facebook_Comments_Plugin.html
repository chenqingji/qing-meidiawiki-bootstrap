Facebook Comments Plugin

This isn't a plugin in the normal context, more a tutorial on how to include Facebook Comments to the bottom of your Mediawiki articles. I also thought that some of you would find this a useful addition because the majority of commenting script are a little large, intrusive and resource intensive. Personally I don't think the Facebook comments plugin is that good looking, but it certainly serves its purpose and it comes with the added bonus of promoting your wiki too.

To install you will need Facebook developer access and only a rudementary understanding of html.

Visit the Facebook Comments page and complete the form - [https://developers.facebook.com/docs/plugins/comments see here] and then click "get code".

If you already have a Facebook api key for your website, ignore the first section of Javascript, but copy and paste the second section (as below) into any article (or location) where you want to display comments. And that is it! See the results below and feel free to post any comments.
