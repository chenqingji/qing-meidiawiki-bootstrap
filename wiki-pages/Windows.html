Scrolling Windows

The new "Scrolling Window" class will be available from version 1.0.45 and as the name suggest displays a small scrollable window for article content, another space saving solution e.g used in an article sidebar panel. If you would like to use Scolling Windows now see below for the guide on how.

Installing Scrolling Windows

bootstrap/bootstrap/css/bootstrap-blue.css (or your chosen colour) and add the following to the bottom of the stylesheet.

/*Windows*/
.window {
    clear: both;
    border-style: solid;
    border-color: #E6E6E6;
    -moz-border-top-colors: none;
    -moz-border-right-colors: none;
    -moz-border-bottom-colors: none;
    -moz-border-left-colors: none;
    border-image: none;
    border-width: 0px 2px 4px;
    border-radius: 4px;
    margin-bottom: 20px;
}
.window > .window-title {
    background-color: #E6E6E6;
    padding: 8px 10px;
    border-bottom: 2px solid #DEDEDE;
    color: #333;
}
.window.window-fill.green {
    border-bottom: 3px solid #138A54;
}
.window.green {
    border-color: #138A54;
}
.window.red {
    border-color: #C62229;
}
.window.blue {
    border-color: #4079B2;
}
.window > .window-body {
    clear: both;
    max-height: 200px;
    padding: 15px;
    overflow: auto;
}
.window.window-fill.green > .window-body {
    background-color: #16A062;
    color: #FFF;
}
.window.window-fill.red > .window-body {
    background-color: #DA282F;
    color: #FFF;
}
.window.window-fill.blue > .window-body {
    background-color: #4C86BF;
    color: #FFF;
}
.window.green > .window-title {
    background-color: #16A062;
    border-bottom-color: #138A54;
    color: #FFF;
}
.window.blue > .window-title {
    background-color: #4C86BF;
    border-bottom-color: #4079B2;
    color: #FFF;
}
.window.red > .window-title {
    background-color: #DA282F;
    border-bottom-color: #C62229;
    color: #FFF;
}
.window > .window-title {
    padding: 8px 10px;
}
