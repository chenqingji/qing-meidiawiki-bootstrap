Twitter Timeline Plugin

Simple to use and install Twitter Timeline plugin for Mediawiki.

Features include links opening in a new window and auto updating, which refreshes every 10 minutes.

Installation

Download the files [http://www.mediawikibootstrapskin.co.uk/forum/download/file.php?id=56 here].

Unpack the zipped folder and move timeline.js to bootstrap/bootstrap/js

Open to edit bootstrapskin.php and add the following to the bottom of the js array

'bootstrap/bootstrap/js/timeline.js',

Visit your wiki and open Mediawiki:Common.js and add the following block of code.

 $('#tweecool').tweecool({
      	 username : 'Bootstrapskin', 
         limit : 5	
      });

And finally add the following to your article to include your Twitter feed;

Configuration

The plugin can also be styled relatively easily by adding a wrap class and then adding css styling dependant on your requirements.

Then add your styling to Mediawiki:Common.css e.g.;

.tweet-wrap li {
   margin-bottom: 10px;
   padding:5px;
   list-style:none;
   clear: both;
   border: 1px solid #E8E8E8;
   border-radius: 5px;
}

.tweet-wrap img {
   float:left;
   margin-right: 10px;
   margin-bottom: 5px;
}

Set options by editing timeline.js

     username : 'tweecool',
     limit : 5,
     profile_image : true,
     show_time : false

Visit [http://www.tweecool.com/ Tweecool] for addition information and advanced options etc.
