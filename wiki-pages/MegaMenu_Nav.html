      YouTube Plugin

      You can add video to BootStrapSkin by installing a plugin.

Megamenu is a brand new feature to BootStrapSkin and will be available as standard in the next update (2015 ?). If you would like to add Megamenu follow the steps below. If you would like to add video as we have done in this demonstration you will need to install the [http://mediawikibootstrapskin.co.uk/235/index.php?title=Bootstrapskin_Youtube Youtube extension].

Installation

Navigate to Mediawiki:Common.css and add;

/* MegaMenu */
.navbar.mega .nav,
.navbar.mega .collapse,
.navbar.mega .dropup,
.navbar.mega .dropdown {
  position: static;
  padding-right: 10px;
  padding-top: 3px;
}
.navbar.mega .container {
  position: relative;
}
.navbar.mega .dropdown-menu {
  left: auto;
}
.navbar.mega .nav.navbar-right .dropdown-menu {
  right: 0;
  left: auto;
}
.navbar.mega .mega-content {
  padding: 23px 23px + 5;
}
.navbar.mega .dropdown.mega-full-width .dropdown-menu {
  right: 0;
  left: 0;
}
.navbar.mega.navbar-inverse .dropdown-menu {
  margin-top: 1px;
  border: 2px solid #1f1f1f;
}
.navbar.mega.navbar-inverse .dropdown-menu > li {
  color: #eee;
  background-color: #4d4d4d;
  border-color: #1a1a1a;
}
.navbar.mega.navbar-inverse .dropdown-menu > li a {
  color: #fff;
}
.navbar-header {
    float: left;
    padding-top: 3px;
    padding-right: 10px;
}
