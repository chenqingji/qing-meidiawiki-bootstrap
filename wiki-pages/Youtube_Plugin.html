Bootstrapskin Youtube Plugin

This is a jQuery plugin to lazy load Youtube videos. On the initial load, the div is replaced by a preview image of the video. On click of the image, the preview image is replaced by the autoplaying iframe Youtube video.

This is very useful because ordinarily Youtube videos cannot be added to Mediawiki content without installing an extension.

Installation

Download the files [http://www.mediawikibootstrapskin.co.uk/forum/download/file.php?id=55 here].

Unpack the zipped folder and move lazyYT.js to bootstrap/bootstrap/js

Open to edit bootstrapskin.php and add the following to the bottom of the js array

'bootstrap/bootstrap/js/lazyYT.js',

Visit your wiki and open Mediawiki:Common.js and add the following block of code.

$('.lazyYT').lazyYT();

Add the following Mediawiki:Common.css .

.lazyYT-title {
    z-index: 100!important;
    color: #fff!important;
    font-family: sans-serif!important;
    font-size: 12px!important;
    top: 10px!important;
    left: 12px!important;
    position: absolute!important;
    margin: 0!important;
    padding: 0!important;
    line-height: 1!important;
    font-style: normal!important;
    font-weight: normal!important;
}
.lazyYT-button {
    margin: 0!important;
    padding: 0!important;
    width: 60px!important;
    height: 41px!important;
    z-index: 100!important;
    position: absolute!important;
    top: 50%!important;
    margin-top: -22px!important;
    left: 50%!important;
    margin-left: -30px!important;
}

And finally add the following to include a video;
