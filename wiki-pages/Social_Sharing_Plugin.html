Social Sharing Plugin

Please to announce the release of a new plugin to the BootStrapSkin Library of addons.

This new addon allows for the addition of social sharing functionality to any location within Mediawiki artcles."


Features

Absolutely no requirement to add links to your content, this is handled by the Javascript, meaning the same block of code can be applied to any page and yet a different page can be shared through any of the social sites.

Limitiations

You will need to create a Facebook App to enable the Facebook sharing feature (see below).

Installation

Download the files [http://www.mediawikibootstrapskin.co.uk/forum/download/file.php?id=54 here].

Unpack the zipped folder and move share.min.js to bootstrap/bootstrap/js

Open to edit bootstrapskin.php and add the following to the bottom of the js array

'bootstrap/bootstrap/js/share.min.js',

Visit your wiki and open Mediawiki:Common.js and add the following block of code.

new Share(".share-button", {
  networks: {
    facebook: {
      app_id: "your facebook app id"
    }
  }
});

Creating a Facebook App

Visit the [https://developers.facebook.com/ Facebook Developers] website to create a new app.

Select the "app" tab and then "create a new app".

Create a display name and namespace, also select a categoty, then click "create app" and complete the security check, wait while your app is created.

Copy the app id at the top of the page and paste the id into the code in Common.js

Now select settings (from the left hand side menu) and click "Add Platform".

Select "website" and add your website url. Note that the url should not contain http or www, but it should include a trailing forward slash i.e. yourwebsite.com/
